LeoStats

LeoStats is a compact & simple dashboard to better inform FIRST Robotics teams about their schedules, placements, alliances, & more during an event.

It's built using JavaScript, both client and server (using Node.Js) side. It gathers its data from the FIRST Events API and sorts that data in ways which are relevant to the end user.

https://github.com/FRC-4459/LeoStats

Calico üêà

The Calico collection is a repository of C++ files which are labeled corresponding to Calico problems.

Calio is a semiannual programming competition held by U.C. Berkely for high school students. These programs are solutions to some of the sample, in-competition, & past programming problems provided by U.C. Berkely. These questions range from simple algorithm-building challenges to use of high-level concepts such as Dynamic Programming, Search/Sorting Algothms, Graphing Algorithms, Shortest Path Algorithms, and more.

https://github.com/kingstonv1/calico

RemindBot ü§ñ

RemindBot is a project which aims to connect LSHS STEM students further with their teachers through connection of the Remind and Discord platforms.

This project will start development in the near future. The goal of this bot is to allow teachers to opt-in to automatically sending their Remind messages to the official LSHS STEM Discord server. The bot achieves this once invited to the Remind by scanning its email for messages posted by teachers on Remind and "Forwarding" their contents to the Discord server using Discord.Js.

https://github.com/kingstonv1/remindBot
